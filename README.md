# Employee Attrition Prediction System

This project is a Machine Learning web application built using **Streamlit** and **Logistic Regression** to predict whether an employee is likely to leave the company or stay.

---

# Project Overview

The application takes employee details such as:

- Age
- Monthly Income
- Job Level
- Work Life Balance
- Job Satisfaction
- Overtime
- Department
- Years at Company
- and more...

Then predicts:

- Employee likely to leave
- Employee likely to stay

using a Logistic Regression model.

---

- Pandas
- Scikit-learn

# Machine Learning Algorithm
